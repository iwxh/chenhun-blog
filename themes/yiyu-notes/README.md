# 晨昏札记 WordPress 主题

这是一个以 Blocksy 为父主题、不依赖页面构建器、图标库、外部字体或第三方 JavaScript 的轻量个人博客子主题。

## 安装前提

- WordPress 6.2 或更高版本
- PHP 7.4 或更高版本
- 已安装 Blocksy 父主题（目录名必须为 `blocksy`）
- 当前目录只是一份候选主题源码；复制到 `wp-content/themes/yiyu-notes` 后，仍需由站点管理员手动预览并启用

## 建议的后台配置

1. 新建“首页”和“文章”两个页面，在“设置 → 阅读”中设置静态首页与文章页。
2. 新建 slug 为 `about` 的“关于我”页面，并选择“关于我”模板。
3. 建立分类：技术折腾、独立站记录、城市生活、摄影设备、AI 工具、随笔札记。推荐 slug 分别为 `tech`、`indie-web`、`city-life`、`photography`、`ai-tools`、`notes`。
4. 在“外观 → 菜单”中配置主导航。
5. 在“外观 → 自定义 → 晨昏札记 · 站点文案”中填写首页文案、公开邮箱、GitHub 用户名和 ICP 备案号。
6. 将希望展示在首页“精选文章”的内容设为置顶文章，最多显示 5 篇。
7. 在“设置 → 固定链接”中选用简洁结构，例如 `/%postname%/`；修改线上结构前先评估旧链接与重定向。

## 回滚

本主题不会自动启用、写入文章或修改数据库。若启用后需要回滚，在后台切回原主题即可；确认不再使用后再删除 `wp-content/themes/yiyu-notes`。
